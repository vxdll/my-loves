# Лендинг-поздравление

## Как менять фото
Положи свои фотографии в папку `assets/photos` и назови их:

- `main-photo.jpg` — главное фото сверху
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`

Можно использовать `.png`, но тогда поменяй расширение в `index.html`.

## Как менять текст
- Основной текст: `index.html`
- 27 причин и фразы по кнопке-сердечку: `script.js`
- Цвета и внешний вид: `style.css`, блок `:root`

## Как выложить на GitHub Pages
1. Создай репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки.
3. Открой `Settings` → `Pages`.
4. В `Branch` выбери `main` и папку `/root`.
5. Через минуту GitHub даст ссылку на сайт.
